## 🌟 Overview

**Elevana** is a premium, AI-powered study companion designed for university students. It transforms dense lecture materials (PDFs, slides, documents, and even audio) into actionable study tools. Stop drowning in readings and start mastering your curriculum with personalized summaries, interactive quizzes, and intelligent study plans.

## ✨ Key Features

- 🧠 **AI-Powered Summarization**: Instantly extract key concepts and high-level summaries from lecture slides and textbook chapters.
- 📝 **Interactive AI Quizzes**: Test your knowledge with MCQ and short-answer questions generated directly from your specific course materials.
- 📅 **Personalized Study Plans**: Receive a tailored study schedule based on your exam dates and learning pace.
- 🎙️ **Multimedia Support**: Analyze digital text (PDF, PPTX, DOCX) and transcribe audio/video lectures to create searchable, structured notes.
- 📊 **Progress Analytics**: Track your mastery of topics with beautiful visualizations powered by Recharts.
- 🔒 **Privacy Focused**: Your data is secured with enterprise-grade encryption and is never used to train public AI models.

## 🚀 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Charts**: [Recharts](https://recharts.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with a custom design system focusing on "Plus Jakarta Sans" and "Inter" typography.

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/abdo-saleh665/Elevana.git
   cd Elevana
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Setup:**
   Create a `.env.local` file in the root directory and add your Gemini API key:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `/pages`: Core application views (Landing, Dashboard, AI Tutor, Quizzes).
- `/components`: Reusable UI elements (Modals, FAQ items, Layouts).
- `/types.ts`: Centralized TypeScript definitions.
- `index.html`: Contains the core design system and Tailwind configuration.
